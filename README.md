# 🚀 CodeCraftHub Learning Management System

A simple, lightweight REST API for tracking your learning goals and courses. Built with Java and Spring Boot.

## ✨ Features
* Complete CRUD operations for course management.
* JSON file-based storage (No database setup required).
* RESTful API design.
* Robust error handling and input validation.

## 🛠️ Installation & Setup
1. Clone or download this project to your local machine.
2. Ensure you have **Java JDK 17** (or higher) and **Maven** installed.
3. Open your terminal, navigate to the project directory, and run:

```bash
mvn clean install
mvn spring-boot:run
